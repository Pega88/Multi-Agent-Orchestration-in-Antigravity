Meet Gemini-CLI

Your intelligent, safe, and powerful AI companion right in the terminal.

Core Capabilities

💻

Code Generation & Editing

Autonomously create, modify, and refactor code across your workspace using advanced search and replace tools.

🔒

Security Audits

Built-in static analysis (SAST) to identify, report, and remediate security and privacy vulnerabilities in your codebase.

🏗️

System Operations

Safely run terminal commands, manage local environments, and orchestrate complex tasks safely with user confirmation.

🧠

Specialized Sub-Agents

Delegates tasks to expert agents for codebase investigation or CLI help, maintaining a modular and effective workflow.

🌐

Google Workspace Integration

Read emails, schedule calendar events, create Google Docs, and interact with Google Drive right from the terminal.

🛠️

Custom Skills

Extend the CLI's capabilities using the Skill Creator to add custom workflows and custom APIs for your organization.
